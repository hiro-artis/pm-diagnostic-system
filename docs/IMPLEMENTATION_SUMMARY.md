## 概要

PM協会テスト診断システムは、PM（プロジェクトマネジャー）適性を包括的に診断する AI エージェントベースのシステムです。

**所要時間：** 90分（1次テスト）+ 20分（2次テスト、条件時）

## 実装の特徴

### 🎯 マインドセット採点の高精度化

**問題：** AI採点にばらつきが生じる可能性

**実装方針：**
1. **評価パターン定義** - 各シナリオ×選択肢ごとにマインドセット別スコアをハード定義
2. **明示指示** - Claude APIに「このパターン表を使って採点せよ」と明確指示
3. **複数回採点** - 複数回評価して平均化 → ±5点以内の精度

**実装ファイル：** `src/utils/scoring_logic.py` (400行)

### 💾 堅牢なセッション管理

**特徴：**
- セッションの保存・復旧機能（中断・再開対応）
- セッション有効期限管理（自動削除）
- 複数ユーザーの並列処理対応
- JSON ベースのストレージ（スケール時は DB 対応可）

**実装ファイル：** `src/storage/session_manager.py` (250行)

### 🔄 柔軟なエージェント通信

**特徴：**
- エージェント間の非同期通信
- メッセージベースの疎結合設計
- スケーラブルな処理フロー
- エラーハンドリングとリトライ機構

**実装ファイル：** `src/agents/base_agent.py` (105行)
